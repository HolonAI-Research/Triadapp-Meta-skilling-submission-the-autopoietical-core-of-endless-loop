# Blind‑Spot Registry (PCN)

## Purpose
To systematically rotate the *blind‑spot* assignment across knowledge
domains, ensuring that the system continuously explores under‑examined
areas and never settles into a static expertise pattern.

## Inputs
- `domain_activity_json`: Path to `400_Knowledge_Topology/domain_activity.json`
  (maps domain IDs to recent artifact counts).
- `blindspot_assignments_md`: Path to `400_Knowledge_Topology/blindspot_assignments.md`.
- `rotation_threshold`: Integer (default 5) – number of new artifacts that
  triggers a rotation.

## Process
1. **Load Activity Log** – Read the JSON to obtain per‑domain counts.
2. **Detect Over‑Populated Domains** – Identify any domain where
   `count >= rotation_threshold`.
3. **Select Next Blind‑Spot** – Choose the domain with the lowest current
   count among those **not** already a blind‑spot.
4. **Update Assignment** – Write the new mapping to
   `blindspot_assignments.md` (maintains historic log of rotations).
5. **Log Rotation** – Append an entry to
   `600_Session_Archives/blindspot_rotations.log` with timestamp,
   old blind‑spot ID, new blind‑spot ID, and the counts.
6. **Issue Gate Flag (optional)** – If a domain has been a blind‑spot for
   more than 10 rotations without change, create/overwrite
   `200_Failure_Signatures/gate.json` with `{ "gate": "block_new_artifacts" }`
   to temporarily pause ingestion from that domain.

## Outputs
- Updated `blindspot_assignments.md`.
- Rotation log entry.
- Optional gate flag file.

## Integration Points
- Called by `memory-integration-and-consolidation` after new artifacts are
  persisted.
- Consumes `domain_activity_json` generated by
  `vault-mining-for-patterns` (or directly by the vault watcher).
- Provides the new blind‑spot identifier to
  `surprise-landscape-navigation` (exploration weighting) and to the
  gate mechanism used by `memory-integration-and-consolidation`.

## Dependencies
- `read_file`
- `write_file`
- `execute_code` (optional to create gate flag)
- `terminal` (optional background logging)
